建設・設備系企業のコーポレートサイト。参考デザインを忠実に再現し、洗練されたアニメーション演出を実装しています。

## プロジェクト概要

**企業名**: 株式会社NEXUS  
**業種**: 建設・設備工事（電気工事、ダクト工事、エアコン工事など）  
**コンセプト**: 豊かで快適な暮らしのために、幸せと信頼を創造する

## 🎬 サービスセクション - アニメーション仕様詳細

### UIデザイン

#### カード構造
```
┌─────────────────────────────────────┐
│ 写真     │  サービス名（太字）      │
│ エリア   │  説明文（2-3行）        │
│ 40%      │  [詳しくはこちら]ボタン │
│          │  60%                    │
└─────────────────────────────────────┘
```

#### デザイン要素
- **背景**: 白 `#fff`
- **角丸**: 10px
- **シャドウ**: `0 2px 12px rgba(0,0,0,0.06)`
- **写真**: object-fit: cover、清潔感のある実写
- **余白**: padding 35px 40px

### アニメーション技術

#### 発火条件
- Intersection Observer API使用
- threshold: 0.2（20%表示で発動）
- 一度だけ実行

#### タイミング
```javascript
1枚目: 0.0秒 (即座)
2枚目: 1.0秒後
3枚目: 2.0秒後
```

#### 方向パターン
```
カード1 [data-direction="left"]  ← 左から120px移動してスライド
カード2 [data-direction="right"] → 右から120px移動してスライド
カード3 [data-direction="left"]  ← 左から120px移動してスライド
```

#### トランジション
```css
初期: translateX(-120px/+120px) scale(0.95) + blur(8px) + opacity: 0
表示: translateX(0) scale(1) + blur(0) + opacity: 1
時間: 0.8s
曲線: cubic-bezier(0.22, 1, 0.36, 1) /* 滑らかで上品 */
```

#### ホバーエフェクト
```css
hover: translateY(-4px) + シャドウ強化
時間: 0.25s ease
```
