Mission・Vision・Value

LR consultingが大切にしている3つの理念

Mission

全ての企業に平等な
採用機会の創出

大企業も中小企業も、スタートアップも、全ての企業に平等な採用機会を提供します。企業規模や知名度に関わらず、それぞれの魅力を最大限に引き出し、最適な人材との出会いを実現することが私たちの使命です。採用における情報格差や機会格差をなくし、全ての企業が優秀な人材を獲得できる社会を目指します。

Vision

より早く、より効率的に、
より良い人材の採用

テクノロジーと人の力を組み合わせ、採用プロセスの効率化と質の向上を同時に実現します。スピードを重視しながらも、企業と候補者双方にとって最適なマッチングを追求。データ分析と経験に基づいた戦略で、より早く、より効率的に、より良い人材との出会いを創出します。

Value

クライアントの課題に
向き合い続ける
プロであり続ける

一時的な解決策ではなく、クライアントの本質的な課題に真摯に向き合い続けます。採用のプロフェッショナルとして常に学び、進化し、最新の知識とスキルを磨き続けることで、クライアントに最高の価値を提供し続けます。

サービス一覧

LR consultingは、採用に関する幅広いサービスを提供しています。

採用コンサルティング

採用戦略の立案から実行まで、トータルでサポート。企業の成長フェーズに合わせた最適な採用計画を提案します。

採用アウトソーシング

採用業務の代行サービス。求人票作成、母集団形成、選考調整など、採用プロセス全体をサポートします。

採用ホームページ制作

企業の魅力を最大限に伝える採用サイトを制作。候補者の心に響くコンテンツとデザインで、応募を促進します。

SNS運用代行

採用広報のためのSNS運用を代行。企業の魅力を発信し、潜在的な候補者へのリーチを拡大します。

人材紹介

企業のニーズに合った優秀な人材をご紹介。豊富なネットワークを活かし、最適なマッチングを実現します。
