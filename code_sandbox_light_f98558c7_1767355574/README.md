# スマナビ（Smile Navigation）- 矯正医検索LP

## 📋 プロジェクト概要

スマナビは、信頼できる矯正医を探すための地域特化型歯科医院検索ポータルサイトのランディングページです。

### ターゲット
- 20代後半〜40代の女性
- マウスピース矯正に興味がある方
- 信頼できる矯正医を探している方

### コンセプト
- 治療費の透明性（月額1,925円〜）
- 初診料・調整料0円
- 24時間WEB予約可能
- 清潔感・信頼感・医療×美容の融合

## 🎨 デザイン仕様

### カラーパレット
- **メインカラー**: #6EC1E4（淡い水色）
- **アクセントカラー**: #ffd700（ゴールド）、#ffe066（黄色）
- **背景**: #e0f2fd → #ffffff（グラデーション）
- **テキスト**: #2c3e50（ダークブラウン〜ブラック）

### タイポグラフィ
- **日本語**: Noto Sans JP（300-900）
- **ロゴ**: Playfair Display（セリフ体、高級感）
- **見出し**: 900 weight、本文: 400-600 weight

### デザイン要素
- 余白多めの洗練されたレイアウト
- 水色グラデーション背景
- 金色バッジで信頼性訴求
- 横長ヒーロー（左テキスト、右画像）
- 若い日本人女性のビジュアル統一

  - 情報ボックス（横並び）:
    - 治療費: 月額 **1,925円**（税込）〜
    - 平均治療期間: **3〜11ヶ月**
  - 特典エリア:
    - 初診料 **0円**（※一部医院は有料）
    - 調整料 **0円**（※治療費に含まれる）

  - 20代後半〜30代前半の日本人女性
  - 白いブラウス、自然な笑顔
  - 背景: 淡い水色

  - 白背景＋グレー枠

- 大きなCTAボタン（光が流れるアニメーション付き）
- 「完全無料」の強調

### 9. 院内ブログ機能
- **トップページ表示**:
  - 最新3件をカード形式で表示
  - アイキャッチ画像 + タイトル + 抜粋
  - 「すべて見る」リンクでブログ一覧へ遷移
  
- **ブログ一覧ページ（blog.html）**:
  - 全記事を3カラムグリッドで表示
  - カテゴリーバッジ、日付、タイトル、説明文
  - カードホバーで浮き上がるアニメーション
  
- **ブログ詳細ページ（blog-detail.html）**:
  - 個別記事の全文表示
  - カテゴリー、日付、タイトル、本文
  - 関連記事（その他の記事）を3件表示
  - 「ブログ一覧へ戻る」リンク

- **記事データ管理（blog-data.js）**:
  - 記事は配列形式で一元管理
  - 新しい記事を追加するだけで自動反映
  - 各記事には以下を含む：
    - id（一意の識別子）
    - title（タイトル）
    - description（抜粋）
    - image（アイキャッチ画像URL）
    - category（カテゴリー）
    - date（公開日）
    - content（本文HTML）

- **デザイン特徴**:
  - 歯科医院向けの清潔感・信頼感
  - 医療広告ガイドライン準拠
  - レスポンシブ対応（PC: 3列、タブレット: 2列、スマホ: 1列）
  - 統一されたカラー・フォント

### 10. フッター
- シンプルなフッターデザイン
- ロゴとコピーライト
