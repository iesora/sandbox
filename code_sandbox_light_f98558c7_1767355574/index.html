歯科検診の保険適用について

症状に応じて、保険が適用されるケースがあります

歯科検診はすべてが自費診療というわけではありません。歯ぐきの腫れや出血、歯石の付着、違和感など、気になる症状がある場合には、歯周病やむし歯の疑いがあると判断され、診察や検査に健康保険が適用されるケースがあります。

一方で、症状がなく健康確認のみを目的とする場合には、自費診療となることもございます。受診の際には、現在のお口の状態や気になる点を歯科医師にお伝えください。症状や診療内容によって保険の適用可否が変わりますので、詳しくは各医院にご確認いただくことをおすすめいたします。

安心して治療を受けていただくために、ご不明な点はお気軽にご相談ください。
