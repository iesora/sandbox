# LR consulting Website

LR consultingの公式ウェブサイトです。「もっと幸せ、もっと豊かに。」をコンセプトに、人と企業の成長を支援するコンサルティング会社のウェブプレゼンスを提供します。

## 🌸 プロジェクト概要

このウェブサイトは、LR consultingの理念、サービス、採用情報、お問い合わせ機能を提供する静的ウェブサイトです。

### 主な特徴

- **モダンで温かみのあるデザイン**: 花をモチーフにした柔らかく親しみやすいビジュアルデザイン
- **レスポンシブ対応**: スマートフォン、タブレット、デスクトップに最適化
- **スムーズなインタラクション**: スクロールアニメーション、ホバーエフェクト、スムーズスクロール
- **アクセシビリティ**: セマンティックHTMLとユーザビリティを考慮した設計

## 📁 プロジェクト構造

```
.
├── index.html          # ホームページ（企業理念・サービス紹介）
├── recruit.html        # 採用情報ページ
├── contact.html        # お問い合わせページ
├── css/
│   └── style.css      # メインスタイルシート
├── js/
│   └── main.js        # JavaScriptインタラクション
└── README.md          # このファイル
```

## 🎯 実装済み機能

### 1. ホームページ (index.html)
- ヒーローセクション（キャッチコピーと装飾）
- 企業理念・フィロソフィーセクション
- コンサルティングテーマの3つのカード
- サービス紹介（3つのサービス）
- 特徴紹介（4つの円形バブル）
- CTA（お問い合わせへの誘導）
- フッター（花のイラストと企業情報）

### 2. 採用ページ (recruit.html)
- パンくずナビゲーション
- 外部採用サイトへのリンク（Green、BIZREACH）
- 公式note紹介セクション
- 個人情報取り扱いに関する説明

### 3. お問い合わせページ (contact.html)
- 3ステップの進行状況表示
- お問い合わせフォーム
  - 名前（必須）
  - メールアドレス（必須）
  - 電話番号（任意）
  - お問い合わせ内容（必須）
- フォームバリデーション
- プライバシーポリシーへのリンク

### 4. インタラクション機能
- スムーズスクロール（アンカーリンク）
- トップへ戻るボタン（スクロール時に表示）
- スクロールアニメーション（要素が表示される際のフェードイン）
- フォームバリデーション
- ヘッダーのスクロール時シャドウ効果
- 現在ページのナビゲーションハイライト

### 5. レスポンシブデザイン
- **デスクトップ（1025px以上）**: 吹き出し4列、横並び配置、重なりなし
- **タブレット（769px-1024px）**: 吹き出し2列、横並び配置、重なりなし
- **スマホ（768px以下）**: 吹き出し縦並び、左右交互配置、背景のみ重ね表示、テキストは常に最前面

## 📝 カスタマイズ方法

### カラーの変更
`css/style.css`の`:root`セクションでCSS変数を変更：

```css
:root {
    --primary-green: #7FB069;
    --primary-yellow: #FFD23F;
    /* 他のカラー変数... */
}
```

### フォントの変更
`body`セレクタの`font-family`プロパティを変更：

```css
body {
    font-family: 'Your Font', Arial, sans-serif;
}
```

### アニメーション速度の調整
各要素の`transition`プロパティで調整可能。

## 🌐 デプロイ方法

このプロジェクトは静的ウェブサイトのため、以下のサービスで簡単にデプロイできます：

1. **Publish タブを使用**: 画面上部の「Publish」タブからワンクリックでデプロイ
2. **GitHub Pages**: GitHubリポジトリにプッシュしてPages機能を有効化
3. **Netlify**: ドラッグ&ドロップで即座にデプロイ
4. **Vercel**: GitHubと連携して自動デプロイ
